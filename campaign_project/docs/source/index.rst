Thendo Website documentation
============================

Welcome to the Thendo Website Documentation!
============================================

This documentation provides an overview of the **Thendo Campaign Website**, built with Django. It covers how to set up the website, its features, and guides for managing campaigns and users.

Overview
--------
The Thendo Campaign Website allows users to create and manage various campaigns. It provides user roles, campaign tracking, reporting, and a management interface.

Key Features:
-------------
- **Campaign Management**: Admins can create and manage campaigns, set deadlines, and assign users.
- **User Roles**: Different user roles such as admin, manager, and user allow for access control and specific permissions.
- **Reports**: Generate and download reports on campaign performance and user activity.
